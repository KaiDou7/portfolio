Goals

Being a music creating software the amount of competetion was expected to be very high, so inorder to stand on top of the competetion I had to design something that was simple to understand and easy to get into, i.e. when the user ventures the website he says "Damn!! That's so Easy". Keeping this in mind and moving forward, I ended up with a very simple logo design with only a circle and a line which perfectly represents simplicity and looks like a computer, re-enforcing the fact that the software is a desktop application.
